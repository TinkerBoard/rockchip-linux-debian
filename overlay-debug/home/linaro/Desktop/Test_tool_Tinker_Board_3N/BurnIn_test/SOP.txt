## BurnIn Test Tool SOP##
1. sudo Burnin.sh $PRODUCT ,ex: TINKERBOARD_3N
2. Enter test case number, and start test

Log will print to terminal directly, also save to /var/log/burnin_test and a link folder /home/linaro/Desktop/burnin_test
ex. cpu.txt,BurnIn.txt......
